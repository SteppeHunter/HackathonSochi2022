## Глоссарий

- **ДБО** - Дистанционное банковское обслуживание
- **ИБ** - Интернет банк (Web сайт ДБО)
- **МБ** - Мобильный банк (Нативное приложение ДБО для мобильных устройств)
- **ОС** - Операционная система (Operational System)
